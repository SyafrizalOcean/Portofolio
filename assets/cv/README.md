# CV File

Upload CV kamu di folder ini dengan nama file **persis**:

```
CV_Syafrizal_Hidayat.pdf
```

Tombol "Download CV" di hero portofolio otomatis link ke file ini. Setelah upload, tombol langsung berfungsi tanpa perlu edit kode.

**Tips**:
- Format PDF (bukan DOCX) agar bisa dibuka di HP recruiter tanpa install Word
- Ukuran ideal: < 2 MB (biar cepat download di koneksi lambat)
- Nama file **jangan diubah** — kalau diubah, tombol jadi broken link
