# Foto & Cover Publikasi

## Cover Buku

Upload cover asli buku "Menjelajahi Laut dan Mengenal Bencananya" dengan nama:

```
book-cover.png
```

Ukuran ideal: **400 × 550 px** (rasio ~1:1.4 seperti sampul buku).

Sebelum di-upload, portofolio akan menampilkan **placeholder SVG** yang mirip cover asli.
