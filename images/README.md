# Panduan Foto Portofolio

## Foto Hero (3 foto berganti-ganti)

Kamu bisa upload **3 foto berbeda** yang akan bergantian sinkron dengan typewriter. Kalau file spesifik belum ada, otomatis fallback ke `profile.png`.

| Slot                       | Nama file                    | Sinkron dengan typewriter    |
|----------------------------|------------------------------|------------------------------|
| Foto 1                     | `profile-software.png`       | Software Engineer            |
| Foto 2                     | `profile-ocean.png`          | Oceanographer                |
| Foto 3                     | `profile-instrument.png`     | Instrumentation Engineer     |
| Fallback (sudah diupload)  | `profile.png`                | (dipakai kalau slot kosong)  |

**Ukuran ideal**: rasio 4:5 portrait (misal 800 × 1000 px). Foto akan cross-fade selama 0.8 detik saat pindah role.

**Ide foto**:
- **Software Engineer**: kamu di depan laptop / dengan monitor coding
- **Oceanographer**: kamu di lapangan pesisir / dengan latar laut
- **Instrumentation Engineer**: kamu dengan alat (CTD, GPS, RTK, sensor)
