# Foto Bidang Keahlian Oseanografi

Upload foto tiap keahlian dengan nama file yang persis sesuai tabel.

| Keahlian                       | Nama file               |
|--------------------------------|-------------------------|
| Remote Sensing                 | `remote-sensing.jpg`    |
| Pemodelan Oseanografi          | `pemodelan.jpg`         |
| Prosesing & Visualisasi Data   | `visualisasi.jpg`       |
| Survey Hidrografi              | `hidrografi.jpg`        |

**Saran foto**:
- Remote Sensing: screenshot citra satelit, peta thermal, atau kamu di depan monitor pengolahan data
- Pemodelan: hasil simulasi model laut, atau visualisasi output SWAN/Comcot
- Visualisasi: dashboard/peta interaktif yang kamu buat
- Hidrografi: kamu di lapangan saat survei batimetri/topografi

**Ukuran ideal**: rasio 4:3 (misal 800 × 600 px).
