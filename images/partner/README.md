# Logo Mitra & Kolaborator

Upload logo tiap mitra dengan nama file yang persis sesuai tabel. Format PNG (latar transparan lebih bagus).

| Mitra              | Nama file           |
|--------------------|---------------------|
| PT Brajakara       | `brajakara.png`     |
| PT Prihaditama     | `prihaditama.png`   |
| ITB                | `itb.png`           |
| KKP                | `kkp.png`           |
| BRIN               | `brin.png`          |

**Ukuran ideal**: 400 × 400 px, latar transparan. Kalau logo bentuknya landscape/horizontal, tetap OK — akan di-fit di dalam kartu.
