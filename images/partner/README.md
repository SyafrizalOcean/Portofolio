**Ukuran ideal**: 400 × 400 px, latar transparan. Kalau logo bentuknya landscape/horizontal, tetap OK — akan di-fit di dalam kartu.

Sebelum logo diupload, kartu akan menampilkan nama mitra sebagai fallback.
